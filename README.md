# Installation Notes

  - Install IDE (Eg: Visual Studio Code).
  - Install Git.
  - Install Node from nodejs.com
  - Clone the Repository.
  - Open project folder (UI_Application) in IDE.
  - Go to IDE commandline and give command "npm install".
  - Now give command "npm start".
  - Team Input App runs...
